解决git push仓库报错

发表于2021-02-10|更新于2021-02-10|工具使用

错误原因

远程库与本地库不一致造成的，那么我们把远程库同步到本地库就可以了。

解决方法

1.重新pull
git pull --rebase origin master
2.重新push
git push origin master
